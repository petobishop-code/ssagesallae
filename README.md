# 싸게살래 쇼핑 가이드 사이트

## 파일
- index.html: 메인 페이지
- style.css: PC/태블릿/모바일 반응형 디자인
- script.js: 검색 UI 기본 동작
- robots.txt / sitemap.xml: 검색엔진 등록용 기본 파일

## 배포 전
1. robots.txt와 sitemap.xml의 YOUR-DOMAIN.co.kr을 실제 도메인으로 교체합니다.
2. 상품 이미지와 상품 링크는 실제 사용 권한이 있는 자료와 승인된 제휴 링크로 교체합니다.
3. 제휴 고지 문구는 이용 중인 제휴 프로그램의 최신 정책에 맞게 최종 확인합니다.

GitHub 저장소에 전체 파일을 업로드한 뒤 Vercel에 연결하면 바로 배포할 수 있습니다.
